Instructions for Claude Code
Project: Synthetic Comment Generation and Semantic Matching System (OpenRouter)
Objective

Build a pipeline that:

Uses OpenRouter LLMs to generate synthetic social-media-style comments from a prompt template.
Compares those generated comments semantically against an existing dataset of real comments.
Returns similarity scores and matched examples from the dataset.

The goal is to evaluate how closely synthetic comments resemble real comments in the dataset.

1. Dataset Format

The dataset consists of rows representing social media comments.

Example row structure:

sentiment_id | tweet_id | timestamp | query | username | comment_text

Example:

0	1467810369	Mon Apr 06 22:19:45 PDT 2009	NO_QUERY	_TheSpecialOne_	@switchfoot http://twitpic.com/2y1zl - Awww, that's a bummer. You shoulda got David Carr of Third Day to do it. ;D

Important column:

comment_text

This is the only column required for semantic comparison.

All other metadata can be ignored unless needed later.
